# `nas_material_url` 填写规则

适用范围：FunnyFuzzy 官网素材插件

## 1. 基本规则

- `nas_material_url` 继续只维护这一列
- 不加标签的路径，默认识别为`图片素材`
- 在路径前加 `video:` ，识别为`视频素材`

## 2. 标准路径格式

请统一填写为 Finder 复制出来的标准格式：

```text
'/Volumes/公司项目/FUNNYFUZZY/设计部/产品/具体文件夹'
```

不要填写以下格式：

- `192.168.2.4/...`
- `smb://192.168.2.4/...`
- 飞书自动识别后的超链接格式

## 3. 图片素材写法

直接填写路径即可：

```text
'/Volumes/公司项目/FUNNYFUZZY/设计部/产品/XXX/图片目录'
```

## 4. 视频素材写法

在路径前加 `video:`：

```text
video:'/Volumes/公司项目/FUNNYFUZZY/设计部/产品/XXX/视频目录'
```

兼容以下写法：

```text
video:'/Volumes/...'
video: '/Volumes/...'
VIDEO:'/Volumes/...'
```

## 5. 同一个产品有多条素材链接

可以继续放在同一个单元格里，用逗号分隔：

```text
'/Volumes/.../图片目录1','/Volumes/.../图片目录2',video:'/Volumes/.../视频目录1'
```

插件显示规则：

- 普通路径 -> 黑色`图片素材`按钮
- `video:`路径 -> 蓝色`视频素材`按钮
- 没有任何路径 -> 灰色`暂未关联NAS，去反馈`按钮

## 6. 建议

- 图片目录不要加 `image:`，默认就是图片
- 只给视频目录加 `video:` 即可
- 路径尽量直接从 Finder 复制，避免手动输入出错
